# Wasvy experiments

## Dependencies

- [rust](https://rust-lang.org/tools/install/)
- [just](https://just.systems/)
- [bacon](https://dystroy.org/bacon/) via `cargo install --locked bacon`

## Development

Get started by running `just start`.

For hot reloading support for both mods and the bevy app you can run `just develop` and `just develop-mods` in two separate terminals.
